# Git Branch Diff Extractor

VSCodeのサイドバーからGitブランチを選択し、そのブランチで変更されたファイルを丸ごと抽出する拡張機能です。

## 機能

- サイドバーにローカルブランチを一覧表示（直近コミット順）
- ブランチをクリックすると、設定済みの出力先（未設定時はデスクトップ）へ日時フォルダを作成
- 分岐元からの差分ファイルをディレクトリ構造ごとコピー
- 文字コード・改行コードを一切変換せずバイナリコピー

## 使い方

1. アクティビティバーの「Branch Diff Extractor」アイコンをクリック
2. ブランチ一覧から対象ブランチをクリック
3. 設定済みの出力先へ日時フォルダが作成される
4. 差分ファイルが自動抽出されます

コミット範囲を指定して抽出する場合は、ブランチのコンテキストメニューから
「コミット範囲を指定して抽出」を選び、コミット範囲と出力先フォルダを指定します。

## 開発・パッケージ

```bash
npm ci
npm run compile
npm run package
```
